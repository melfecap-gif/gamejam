# StreetWise: The Tale of Two Rule-Breakers 🚦📱

StreetWise is a 3D educational simulator built for the **GameJam 2024**. It uses a dual-narrative approach to teach traffic safety by experiencing the consequences of poor decisions through two "anti-heroes": Dash (reckless driver) and Step (distracted pedestrian).

## 🌟 Visual Style
- **Art Direction**: "Friendly Low-Poly." Rounded edges, vibrant colors, expressive models.
- **Environment**: Clean, modular "Toy Town" layout.
- **Tone**: Encouraging, upbeat, and educational.

## 🕹️ Core Mechanics
- **The Driver’s Path (Dash)**: Third-person driving physics. Focus on signaling, speed limits, and yielding.
- **The Pedestrian’s Path (Step)**: Third-person walking. Focus on crosswalks, signals, and blind spots.
- **Awareness Meter**: Your score depends on your adherence to traffic laws.

## 🛠️ Technical Stack
- **Engine**: Three.js (Web-based 3D) & Cannon-es (Physics).
- **Logic**: Vanilla Javascript (ESM).
- **Styling**: Vanilla CSS with Glassmorphism.
- **Translations**: i18next (English & Portuguese).

## 🚀 Getting Started
1. Clone the repo.
2. Run `npm install`.
3. Run `npm run dev` to start the local development server.
4. Open the displayed URL in your browser.

## 📖 Narrative Chapters
- **Intro**: Dash and Step's reckless start.
- **Chapter 1**: The Right of Way (Dash & Stop Signs).
- **Chapter 2**: The Invisible Pedestrian (Step & Crosswalks).
- **Chapter 3**: Sharing the Road (Combined Mission).
